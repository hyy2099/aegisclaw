# 🦞 金甲龙虾 - 真实操作指南

## 📋 前置准备

### 1. 配置 API Key

如果你还没有币安 API Key：
1. 登录币安官网
2. 进入 API 管理
3. 创建 API Key
4. **重要**：
   - 只启用"现货交易"权限
   - 禁用"提现"权限
   - 绑定 IP 白名单（可选）

### 2. 设置网络环境

**测试网（推荐）**：
```bash
# 编辑 .env 文件
BINANCE_TESTNET=true
```

**主网**：
```bash
# 编辑 .env 文件
BINANCE_TESTNET=false
```

## ⚠️ 重要提醒

### 1. 安全设置
- 使用子账户（推荐 500-1000 USDT）
- 关闭所有危险权限
- 禁用提现权限
- 绑定 IP 白名单

### 2. 风险控制
- 单日交易次数限制：10 次
- 最大价格滑点：1%
- 资金费率套利阈值：0.01%
- 最小套利利润：0.1%

### 3. 盈利模式
- 零钱兑换（Dust Sweeper）：自动将小额资产换成 BNB
- Launchpool：监控新币挖矿，自动申购
- 资金费率套利：现货与合约之间的无风险套利

## 🎯 操作建议

### 新手阶段
1. 只执行 **安全检查** 和 **资产扫描**
2. 不要手动执行交易
3. 观察几次套利机会后再决定是否执行

### 进阶阶段
1. 执行零钱兑换
2. 尝试小额资金费率套利
3. 参与 Launchpool 新币申购

## 🔧 故障排查

### 连接失败
1. 检查 API Key 是否正确
2. 确认网络能访问 api.binance.com
3. 尝试使用测试网
4. 设置代理（如果需要）

### 权限错误
1. 确保禁用了"提现"权限
2. 只保留"现货交易"权限
3. 使用子账户

### 余额不足
- 零钱兑换：需要至少 10 USDT 等值资产
- 套利操作：建议每笔不超过账户的 5%
- Launchpool：最低 10 USDT

## 💡 实战技巧

1. **监控时机**：资金费率通常每 8 小时支付一次
2. **分散风险**：不要把所有资金投入单一策略
3. **及时止损**：如果套利收益低于预期立即停止
4. **定期报告**：每周查看收益统计，优化策略
